# ✈️ Vacation Itinerary Planner

A beautiful and modern vacation itinerary planning application built with React and deployed on Vercel.

**🌍 Language:** The app interface is in Romanian, but this README is in English.

## Features

✅ **Multi-day Planning** - Create and manage multiple vacation days  
✅ **Category Organization** - 🛫 Transport, 🏨 Accommodation, 🍽️ Food, 🎭 Activities, 📌 Other  
✅ **Price Tracking** - Track expenses per item, per day, and total trip cost  
✅ **Location Links** - Add links to hotels, restaurants, attractions  
✅ **Auto-save** - LocalStorage keeps your data safe across sessions  
✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile  
✅ **Modern UI** - Beautiful gradient design with smooth animations  

## Quick Start

### Local Setup

```bash
cd Desktop/todolist
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

### Deploy to Vercel

1. Fork or clone this repo
2. Connect your GitHub repo to Vercel at [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite and deploys automatically
4. Your app is live in ~2 minutes! 🚀

## How to Use

1. **Add Travel Days**: Click "+ Adaugă Zi" to create days for your vacation
2. **Switch Between Days**: Click day buttons to navigate
3. **Add Activities**: 
   - Enter activity name
   - Select category (Transport, Accommodation, Food, Activities, Other)
   - Add price in EUR
   - Optional: paste link to booking/website
4. **Track Costs**: See daily total and full trip budget
5. **Check Off**: Mark items as complete with checkboxes

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool (super fast!)
- **CSS3** - Modern styling with gradients and animations
- **LocalStorage API** - Client-side data persistence

## Data Privacy

✅ Your vacation data is stored **locally on your device** only  
✅ No data is sent to any server  
✅ Clear browser cache = clear data (be careful!)

## Customization

Want to modify the app?

1. Edit `src/App.jsx` for functionality
2. Edit `src/App.css` for styling
3. Edit `src/main.jsx` or `index.html` for structure
4. Push changes: `git push origin main`
5. Vercel redeploys automatically!
